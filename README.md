# 🏗️ Minecraft Server Script

> An example code for your minecraft server.

**⚠️ IMPORTANT:** You need to change all mentions of "GOLD SMP" to your server name.
**Ignore comments in russian or delete them.

---

## 📋 Requirements

| Plugin | Version |
|--------|---------|
| Skript | 2.15.4+ |
| SkBee  | 3.25.1+ |
| Nerves | Latest  |

---

## 🚀 Setup

| Step | Action |
|------|--------|
| 1    | Download **Skript** 2.15.4+ |
| 2    | Download **SkBee** 3.25.1+ |
| 3    | Setup your server |
| 4    | Download `script.sk` |
| 5    | Insert script in `/plugins/skript/scripts/` |
| 6    | Launch server |
| 7    | ✅ Done! |
